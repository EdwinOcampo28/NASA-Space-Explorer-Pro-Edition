# 🚀 Explorador Espacial NASA Pro

## Características
- Imagen Astronómica del Día (APOD)
- Buscador de imágenes espaciales
- Diseño moderno y responsive
- Arquitectura modular en JavaScript

## Tecnologías
- HTML5
- CSS3
- JavaScript Vanilla
- APIs públicas de NASA

## Cómo usar
1. Abre el archivo index.html en tu navegador.
2. Selecciona una fecha para cargar la imagen del día.
3. Busca imágenes del espacio por palabra clave.

Proyecto ideal para portafolio.
